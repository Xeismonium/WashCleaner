<task type="auto">
  <name>Task 1: Implement Firestore Security Rules</name>
  <files>firestore.rules</files>
  <action>
    Implement RBAC logic in firestore.rules per D-01 and SEC-01/SEC-02.
    - Define helper functions: `isAuthenticated()`, `isOwner()`, `isActiveUser()`.
    - `isOwner`: Checks if user document has `role == 'OWNER'`.
    - `isActiveUser`: Checks if user document has `isActive == true`.
    - Apply rules to `users`, `customers`, `services`, and `orders` collections.
    - Orders should only be readable/writable by authenticated active users.
    - Services should be writable only by Owners.
  </action>
