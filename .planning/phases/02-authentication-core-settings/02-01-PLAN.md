| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01-01 | Spoofing | Auth Flow | mitigate | Leverage Firebase Auth standard identity verification (D-01). |
| T-02-01-02 | Tampering | Session Data | mitigate | Use DataStore (private app storage) and verify isActive status on every app start (D-03). |
| T-02-01-03 | Elevation | Role Assignment | mitigate | Roles stored in Firestore, protected by Security Rules (implemented in Plan 03). |
